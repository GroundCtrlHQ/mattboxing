# Home Page

## Purpose
The Home Page serves as the main landing page for "The Boxing Locker" platform, providing users with an overview of Matt Goddard's expertise and easy navigation to all key features of the application.

## Key Features

### Hero Section
- **Professional Credentials Display**: Showcases Matt Goddard's 7-0 professional record and National Championship status
- **Brand Identity**: Prominent "The Boxing Locker" branding with boxing red accent color
- **Value Proposition**: Highlights 20+ years of ring experience

### Interactive Feature Showcase
- **MattFeatureShowcase Component**: Interactive component that demonstrates key platform capabilities
- **Visual Engagement**: Draws users into the platform experience

### Main Feature Cards
Three prominent feature cards that link to core functionalities:

1. **Voice Coach Card**
   - Real-time voice coaching with Freya Mills
   - Direct link to `/aivoice` page
   - Visual indicator with microphone icon

2. **AI Coach Card**
   - Personalized training plan generation
   - Direct link to `/coach` page
   - Premium gold accent for lead magnet

3. **Chat Card**
   - Text conversation with Matt's AI
   - Direct link to `/chat` page
   - Blue accent for conversational interface

### Features Grid
- **600+ Videos**: Showcases comprehensive video library
- **Pro Experience**: Highlights professional credentials
- **Form-Driven**: Emphasizes biomechanics and "Value of Looks" philosophy

### Five Boxing Philosophies
- **Visual Display**: Grid showcasing Brain, Legs, Hands, Heart, and Ego
- **Core Philosophy**: Represents Matt's teaching methodology

## Navigation
- **Sticky Header**: Always accessible navigation bar
- **Responsive Design**: Mobile-first approach with collapsible menu
- **Quick Access**: Direct links to Videos, Voice, Coach, Chat, and About pages

## Design Elements
- **Color Scheme**: Charcoal black background with boxing red and champion gold accents
- **Typography**: Bold, modern sans-serif for headings
- **Mobile Responsive**: Optimized for all screen sizes

## User Flow
1. User lands on home page
2. Sees Matt's credentials and value proposition
3. Explores feature cards to understand platform capabilities
4. Navigates to specific feature (Voice, Coach, or Chat)
5. Can browse videos or learn more about Matt

## Technical Implementation
- **Next.js App Router**: Server-side rendered for SEO
- **Component-Based**: Modular React components
- **Performance Optimized**: Fast loading with optimized images
